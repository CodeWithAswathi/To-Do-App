To-Do App

A simple and user-friendly **To-Do Application** built using **HTML, CSS, and JavaScript**.  
This app helps users manage their daily tasks by allowing them to add, edit, delete, and mark tasks as completed.

Features

✅ Add new tasks  
✅ Mark tasks as completed  
✅ Delete tasks  
✅ Clear all tasks option (optional)  
✅ Responsive design  
✅ Simple and clean UI  
✅ Stores tasks (if localStorage is used)

Technologies Used

- **HTML** – Structure of the app  
- **CSS** – Styling and layout  
- **JavaScript** – Logic and functionality
